# GrabMail

> GrabMail is a simple tool for crawling the available mails in a web page.

## Instalation

### clone
```
git clone https://github.com/brokenheartz/EmailFinder.git
```

### usage
```
$ cd EmailFinder
$ pip3 install colorama requests tld
$ mkdir -v saved/
$ chmod +x main.py
$ ./main.py
```
